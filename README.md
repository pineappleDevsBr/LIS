## Requisitos
* Node >= 12.0.0
* NPM >= 6.0.0
* VueCLI
* AdonisCLI
* MySQL

---

## Setup

### Front-End

Faça uma copia do .env.example para .env colocando as credenciais.

```
cd site
npm install
```

### Back-End

Faça uma copia do .env.example para .env colocando as credenciais.

```
cd api
npm install
```
